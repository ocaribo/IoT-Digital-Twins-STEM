 Physics Scenario – Pendulum Experiment

Learning Objective
Understand oscillatory motion and periodic behavior.

Materials
- Pendulum setup
- IoT sensor
- Microcontroller

 Procedure
1. Assemble the pendulum system
2. Start data acquisition
3. Observe real-time visualization

Digital Twin Interaction
Sensor data is transmitted to the virtual interface for visualization.(In Processing interface)

Expected Outcome
Students relate physical motion to graphical representation.
